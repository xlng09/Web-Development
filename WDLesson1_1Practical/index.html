Web Technologies

HTML

HIML stands for Hyper Text Markup Language.
It is a coding language taht display content on a web page.
It is not a program language.

CSS

CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
CSS saves a lot of work.

JavaScript

JavaScript is a high-level, multi-paradigm programming language primarily known for enabling interactive and dynamic content on websites.
It is one of the core technologies of the World Wide Web,
alongside HTML and CSS.

JQuery

JQuery is a fast, small, and feature-rich JavaScript library designed to simplify client-side scripting of HTML.
It is not a programming language itself, but rather a tool that makes it easier to use JavaScript on websites.
